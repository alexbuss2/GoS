# Project Summary
BİRİKİO is a premium financial tracking assistant aimed at helping users manage their assets intelligently, including gold, cryptocurrencies, stocks, and currencies. The application features a modern interface and offers tools for asset management, price alerts, transaction history, user settings, and a new Freemium subscription model providing enhanced features for users.

# Project Module Description
The project consists of the following functional modules:
- **Authentication**: User login, logout, and session management.
- **Asset Management**: Add, edit, delete, and categorize assets.
- **Price Alerts**: Set and manage price alerts for various assets (available for PRO users).
- **Transaction History**: View past transactions and overall financial performance.
- **User Settings**: Manage personal preferences such as currency, theme, and PIN security.
- **Subscription Management**: Manage user subscriptions and payment processing through Stripe.

# File Description Inventory
- **app/backend/**: Contains backend application code including routing, models, and services.
- **app/frontend/**: Contains frontend application code including components, pages, and styles.
- **README.md**: Documentation for both frontend and backend setup and usage.
- **requirements.txt**: Lists Python dependencies for the backend.
- **package.json**: Lists JavaScript dependencies for the frontend.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Vite
- **Database**: PostgreSQL
- **State Management**: React Query
- **Styling**: Tailwind CSS
- **Payment Processing**: Stripe

# Usage
1. Install dependencies:
   - For backend: `pip install -r requirements.txt`
   - For frontend: `npm install`
2. Build the frontend application:
   - Run `npm run build` in the frontend directory.
3. Start the backend server:
   - Run `uvicorn app.backend.main:app --reload` in the backend directory.
